# 2. ❗ Problem Description

Managing Hajj pilgrims manually is a difficult and time-consuming task for travel agencies and organizers. Maintaining records of pilgrims, packages, hotel bookings, payments, and emergency contacts through paperwork can lead to data loss, delays, and inefficiencies.

This project aims to develop a **smart web-based Hajj Management System** that digitizes the entire management process and provides a centralized platform for handling registrations, packages, payments, and pilgrim tracking.

The system will help organizers efficiently manage Hajj-related information using a modern web application integrated with a **cloud NoSQL database**.

---

# 3. 🧭 System Overview

The **Smart Hajj Management & Pilgrim Tracking System** is a web-based application developed using Flask and MongoDB Atlas.

The system will allow administrators to:

* Register pilgrims
* Manage Hajj packages
* Track payments
* Store emergency information
* Generate QR codes for pilgrims
* Search and filter records
* View dashboard analytics

The frontend will be developed using **HTML, CSS, JavaScript, and Bootstrap**, while Flask APIs will handle backend logic and database connectivity.

**MongoDB Local Server (NoSQL Database)** will be used as the cloud NoSQL database to store and manage all pilgrim-related data.
